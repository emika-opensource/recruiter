# Recruiter — Emika AI Employee

AI-powered hiring pipeline, candidate management, and scoring platform.

## Features
- **Kanban Pipeline** — Drag-and-drop hiring stages with visual candidate cards
- **Candidate Scoring** — Weighted multi-criteria scoring with breakdown visualization
- **Role Management** — Define roles with skills, qualifications, and candidate profiles
- **Platform Integrations** — Greenhouse, Lever, Workable, BambooHR, Ashby, SmartRecruiters (API) + LinkedIn, Indeed, Glassdoor, AngelList (BrowserBase)
- **Onboarding Wizard** — Guided multi-step setup experience
- **Export & Reports** — CSV export, per-role analytics, pipeline metrics

## Quick Start
```bash
npm install
node server.js
# Open http://localhost:3000
```

## Stack
- Express.js backend with JSON file persistence
- Vanilla JS SPA frontend (no framework dependencies)
- DM Sans typography, dark theme (#06060a), amber accent (#f59e0b)
